get_personalized_ranking/2

get_personalized_ranking(Client, Input) -> any()

Re-ranks a list of recommended items for the given user.

The first item in the list is deemed the most likely item to be of interest to the user.

The solution backing the campaign must have been created using a recipe of type PERSONALIZED_RANKING.

get_recommendations/2

get_recommendations(Client, Input) -> any()

Returns a list of recommended items.

The required input depends on the recipe type used to create the solution backing the campaign, as follows:

  • RELATED_ITEMS - itemId required, userId not used

  • USER_PERSONALIZATION - itemId optional, userId required

Campaigns that are backed by a solution created using a recipe of type PERSONALIZED_RANKING use the API.
